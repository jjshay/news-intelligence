## [1.0.0] - 2025-01-11

### Added

- Multi-AI news article scoring using ChatGPT, Claude, Gemini, Grok, and Perplexity
- Individual AI evaluation on factual accuracy, source credibility, relevance, bias, and writing quality
- Peer pairing system for AI models to compare and discuss their scores
- Perplexity integration as fact-checker with real-time web search verification
- 6th AI consolidator call that summarizes all opinions into consensus bullet points
- Health check system to verify all AI APIs are operational before processing
- Weighted scoring based on audience preferences and target demographics
- Google Sheets integration for automatic result storage
- GUI version with visual progress tracking
- News processing utilities for article ingestion
- Sample articles JSON for demonstration
- Demo mode that works without API keys
- Visual showcase script for quick demonstration
- Comprehensive test suite for scoring logic
- CI workflow for automated testing
